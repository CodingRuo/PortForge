PortForge

Ein kleines, selbst gehostetes PaaS (Platform as a Service), das Deployments einfacher machen soll.

Gebaut mit Vue.js, Node.js, Docker und dem Ziel, den Prozess vom Code zur laufenden Anwendung zu automatisieren.

A small, self-hosted PaaS (Platform as a Service) intended to simplify deployments.

Built with Vue.js, Node.js, Docker aiming to automate the process from code to running application.
